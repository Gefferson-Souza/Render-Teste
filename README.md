  # Access
  #### /api/notes
  All notes;
  #
  #### /api/notes/id
  Info about one specific note;
  #
  #### /api/users
  All users information;
  #
  #### /api/users/userid
  All info about a specific user and all his Notes;
  
  # Login
  #### /api/users   
  Make a post request in JSON format 
  ##### Ex: {username: 'test123', password: '123456'}
  #
  #### /api/login 
  After creating a new user, make a post request to log in;
 
  ### Note
All the data are in json format, if you are using google chrome install JSONVue extension to have a better experience or just use Firefox.
